# Learn Helper Frontend

Next.js frontend для Learn Helper.

## Что делает frontend

- показывает экран авторизации;
- загружает источники в backend;
- открывает библиотеку материалов;
- запускает открытие или пересборку mdBook;
- показывает mdBook в iframe;
- содержит вкладку тренажера;
- управляет AI-помощником, генерацией вопросов и проверкой ответов.

Конспект раздела и TTS-кнопка внутри mdBook подключаются backend'ом как дополнительные mdBook assets.

## Запуск

```bash
npm install
npm run dev
```

Приложение открывается на `http://127.0.0.1:3000`.

В dev-режиме `/api/*` проксируется на backend `http://127.0.0.1:4173`, а `/api/tts/*` - на TTS `http://127.0.0.1:8000`.

## Проверка

```bash
npm run lint
```
